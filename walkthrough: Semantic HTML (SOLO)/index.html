My HTML Cheat Sheet

Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one 'h1'.

Paragraphs

Paragraphs of text go in 'p' tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in a 'li' tag.

Types of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in a 'li' tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solutions

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'hfref' attribute. Go to DuckDuckGo to search for more information.

Settings 'target="_blank"' will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page.

Tables are created using the 'table' tag. Tables have up to three parts, 'thead', 'tbody', and 'tfoot'. Within each section use 'tr' tags to create rows. Within those, use 'th scope="row or col"'' to make headers on in each space, or 'td' for data.

Article

'article' represents a self=contained composition in a document, which is intended to be independently distributable. i.e., forum post, newspaper article

Footer

'footer' Is used to make a footer for the nearest ancestor sectioning content or root element.

Header

'header' creates a header that is often used as a navigation aid or contain logos, search forms, etc.

nav

'nav' is used to provide navigation links
